## Project: ClipKeep - Smart Clipboard Manager

Built a clipboard history manager that auto-categorizes copied content.

### What it does:
- Monitors clipboard in background daemon mode
- Auto-categorizes: URLs, emails, phone numbers, IP addresses, hex colors, code snippets, addresses, plain text
- SQLite storage with full-text search
- CLI interface for search, browse, and management

### Tech stack:
- Node.js + clipboardy + better-sqlite3 + commander
- SQLite FTS5 for fast search
- Pattern matching + heuristics for categorization

### Commands:
- `clipkeep daemon` - Start monitoring
- `clipkeep search <query>` - Search history
- `clipkeep recent` - Show recent items
- `clipkeep categories` - List categories
- `clipkeep clean --days 30` - Remove old items
